Drop your app icon PNGs here and keep the names from Contents.json:
- AppIcon-16.png
- AppIcon-32.png
- AppIcon-64.png
- AppIcon-128.png
- AppIcon-256.png
- AppIcon-512.png
- AppIcon-1024.png

Tip: A single 1024×1024 PNG (AppIcon-1024.png) works; Xcode will scale it for all slots. For best results, export all sizes.

After adding files, set the target’s App Icon to "AppIcon" in:
Xcode → Target UpNow → Build Settings → Assets → App Icons Source.
